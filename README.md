# Customer-Segmentation-Retention-Analysis
Machine Learning project using RFM Analysis and K-Means Clustering to identify high-value customers and predict churn risk.
I developed this project to solve a core business problem: Customer Churn;By analyzing transactional data, I used Machine Learning to group customers based on their behavior, allowing the business to target them with specific retention strategies.

*Tech Stack
>Python** (Pandas, NumPy)
>Machine Learning: K-Means Clustering
>Visualization:Matplotlib, Seaborn
>Business Framework:RFM (Recency, Frequency, Monetary) Analysis

*Business Impact & Retention Strategy
>Based on my model, I identified 4 key segments:
1. Champions: High-value loyalists. Strategy: VIP perks and early access.
2. At-Risk:** High spenders who haven't bought recently. Strategy: 20% win-back discount.
3. Hibernating: Low engagement customers. *Strategy: Automated email re-engagement.
4. New Customers:Just started buying. Strategy: Onboarding support.
